# Ethereum Vertically Integrated Stack
Complete Ethereum development environment including private network, block explorer, wallet and smart contracts.
## Quick Start
See [blockcahin/README.md](blockchain/README.md) for setup instructions.
## Components
- **blockchain/** - Private Ethereum testnet (Clique PoA)
- **backend/** - Go services (indexer + API)
- **frontend/** - Next.js applications
- **contracts/** - Solidity smart contracts
